MongoDB 測試 URL 頁面

這個頁面包含你剛剛插入 MongoDB 的五筆測試 URL，可用來測試 Chrome Extension 的 hover tooltip。

  • Real

    https://example.com/news/real-1
  • Fake

    https://example.com/news/fake-1
  • Suspicious

    https://example.com/news/suspicious-1
  • Trusted

    https://trusted.org/article/100
  • Unknown

    https://unknown-site.net/post/999

測試方式：用本機 server 開啟這個檔案，例如 py -m http.server 8000，再進入對應網址測試滑鼠懸停。
